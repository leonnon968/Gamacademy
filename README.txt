This is the demo application for Gamacademy for CSE 682 Fall 2017.

To run the application on local host:
1. Make sure node.js is installed. https://nodejs.org/en/
2. Make sure Google Firebase CLI is installed. Open terminal and type "npm install -g firebase-tools".
3. Navigate to project root folder.
4. Type "firebase login".
5. Type "firebase serve"
6. Go to localhost on web browser.
